1. Install the app

Install from YouTrack Marketplace or upload manually to your YouTrack Server instance.

2. Configure S3 storage

All S3 connection parameters are stored in a single S3 Connection String. To generate it:

  1. Open Administration → Applications → Adv.Attachments → Settings
  2. Scroll to the "Generate S3 Connection String" section
  3. Fill in your S3 connection details:
    • Endpoint — e.g., https://s3.eu-north-1.amazonaws.com or http://localhost:9000
    • Region — e.g., eu-north-1 (use us-east-1 for MinIO)
    • Bucket — your S3 bucket name
    • Access Key ID and Secret Access Key
    • Force Path Style — enable for MinIO (required)
    • S3 Provider — aws_default for AWS, minio_default for MinIO
  4. Click "Generate" — the secured string is created
  5. Paste it into the "S3 Connection String" field and click "Save"

3. Test configuration

Click "Run S3 Healthcheck" button in the Admin panel to verify all operations (list, upload, download, delete) work correctly.

4. Start using

Navigate to any issue — the Adv.Attachments widget appears in the issue footer. You can:

  • Upload files — drag & drop or click "Attach" button
  • Browse files — switch between Grid and List views
  • Preview files — double-click to preview images, PDF, Office (XLSX, DOCX, ODT), JSON, Markdown, ZIP and text files
  • Download / Delete — hover over a file for action buttons

Note: Both native YouTrack attachments and S3 files are displayed together in a unified view. S3 storage is optional — native attachments work without any configuration.

5. CORS setup (important for S3)

If using S3 storage, configure CORS on your S3/MinIO bucket to allow browser access. See setup documentation for details.
